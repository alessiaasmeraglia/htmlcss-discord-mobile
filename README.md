# Discord Mobile

Riproduzione del layout **Discord Mobile** a partire dallo screenshot fornito nell’esercizio.

## Obiettivo

Realizzare una pagina web in **HTML** e **CSS** che replichi il layout mobile di Discord, lavorando con una **larghezza fissa di 512px**.

## Tecnologie utilizzate

* HTML5
* CSS3
* Google Fonts

  * **DM Sans**
  * **Sigmar**
* Bootstrap Icons

## Struttura del progetto

* `index.html` → struttura della pagina
* `style.css` / `index.css` → stile della pagina
* `assets/` → immagini e video forniti per l’esercizio

## Contenuto della pagina

La pagina è composta da:

* **Hero section** con banner iniziale
* **Titolo principale** e testo introduttivo
* **Call to action buttons**
* **Due card principali** con:

  * icona decorativa in alto
  * titolo
  * descrizione
  * contenuto multimediale

## Bonus realizzati

* Icona delle card sempre centrata in alto
* Sfondo delle card con **gradiente**
* Inserimento di **video autoplay, muted e loop** al posto delle immagini statiche

## Scelte di layout

* Pagina centrata orizzontalmente
* Larghezza fissa di **512px**
* Utilizzo di `flex` per l’allineamento verticale degli elementi
* Uso di `position: absolute` per il posizionamento delle icone decorative delle card
* Uso di `border-radius` e gradienti per avvicinarsi il più possibile al design originale

## Note

Progetto realizzato come esercizio di replica layout **Discord Mobile**.
